Privacy Policy

Last updated: May 2026

This app is a static calculator and graphing tool. It does not collect personal data on any server, and it does not send usage data to third parties.

What data is stored

  • Graph formulas entered in the browser.
  • Viewport settings such as x/y range and zoom level.
  • Display preferences including theme and high-contrast mode.
  • Calculator history and last answer value.
  • Local memory values stored in the calculator interface.

How data is stored

All data is stored locally in your browser using localStorage. It remains on your device and is not transmitted to any external server or analytics provider.

No third-party tracking

This page does not use cookies, analytics scripts, advertising, or any third-party tracking services.

Deleting stored data

You can remove saved state by clearing browser storage for this site. This will reset expressions, preferences, history, and local memory.

Updates

We may update this policy if the app changes. The current version will always be available from the site footer.
